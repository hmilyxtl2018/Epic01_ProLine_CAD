# 机身壁板自动钻铆工艺规程 SOP-FSP-002-B

**文件编号**: SOP-FSP-002-B  
**版次**: Rev.2  
**生效日期**: 2025年10月1日  
**适航批准**: CAAC-AP-2025-FSP-002  
**NADCAP认证**: 铆接工艺 AC7114  

---

## 1. 适用范围

本规程适用于C919前机身段（Section-41）壁板组件的自动钻铆装配。包括蒙皮与长桁、蒙皮与框段的铆接连接。材料组合为2024-T3铝合金蒙皮+7075-T6铝合金长桁。

## 2. 引用标准

- HB 5277 航空用铆钉铆接质量检验
- BAC 5583 Boeing铆接标准（参考）
- GJB 715 航空结构铆接通用工艺规范

## 3. 设备要求

### 3.1 自动钻铆机
- Electroimpact E6000 自动钻铆系统（或同等级）
- 铆接力范围：5kN~80kN
- 定位精度：±0.1mm
- **单台峰值功率：150kW**
- **设备占地：8m × 4m（含操作空间）**

### 3.2 辅助设备
- 壁板定位型架（工装号：JIG-FSP-41-R01）
- 铆钉自动送料系统
- 密封胶涂敷机器人（Fanuc M-20iD）
- **激光跟踪仪（Leica AT960）定位精度 ≤ 0.015mm/m**

## 4. 环境要求

- **环境温度**：18°C ~ 28°C
- **环境湿度**：30% ~ 70% RH
- **噪声防护**：铆接区噪声 ≥ 95 dB(A)，操作人员必须佩戴耳塞
- 照明：≥750 lux（铆接区局部照明）
- **约束：自动钻铆机振动会影响相邻精密设备，钻铆机周围8m范围内不可布置CMM或激光跟踪仪等精密测量设备**

## 5. 工艺流程

### 5.1 壁板预装配
1. 将蒙皮铺放在定位型架上
2. 按照技术条件放置长桁，用临时紧固件固定
3. 使用激光跟踪仪检查相对位置
4. **约束：蒙皮与长桁贴合间隙 ≤ 0.3mm（铆接前）**

### 5.2 密封胶涂敷
1. 在铆接界面涂敷PR-1776 B-2密封胶
2. 涂敷厚度：0.1mm ~ 0.3mm
3. **约束：密封胶涂敷后必须在4小时内完成铆接**（超时密封胶固化失效）
4. **约束：密封胶存储温度 -20°C ~ 0°C（冷库），取出后室温回温30分钟方可使用**
5. **约束：密封胶配制间必须具备独立排风系统，与主装配区保持负压隔离**

### 5.3 自动钻铆
1. 自动钻铆机程序加载（NC程序编号与件号对应）
2. 铆钉规格选择：
   - MS20470AD（实心铆钉）Φ4.0~Φ5.0
   - 铆接间距：按图纸标注（典型值 25mm~30mm）
3. 钻铆参数：
   - 钻孔转速：8000 RPM
   - 进给速度：0.05 mm/rev
   - 铆接压力：根据铆钉直径自动调节
4. **约束：铆钉墩头直径 = 1.5D ± 0.1mm（D为铆钉杆径）**
5. **约束：铆孔同轴度 ≤ 0.05mm**
6. **约束：连续100个铆钉中不良率 ≤ 1%，否则需停机检查**

### 5.4 铆接质量检测
1. 目视检查：100%逐铆检查（外观）
2. 铆钉墩头高度测量：30%抽检
3. **约束：超声C扫描检测（壁板组件完成后）覆盖率 100%**
4. **约束：NDT检测需在铆接完成后24小时内进行**

### 5.5 密封胶固化
1. 铆接完成后在铆缝处补涂密封胶
2. 室温固化：48小时（完全固化）
3. 加速固化（可选）：60°C 烘箱 4小时
4. **约束：固化期间零件不可移动或施加外力**

## 6. 刀具与消耗品管理

- 钻头寿命：每500孔更换（或每次降级≥0.02mm时）
- **约束：铆钉批次需与件号关联，可追溯至每架次**
- 密封胶有效期管理：开封后使用期限8小时，过期废弃

## 7. 安全要求

- **约束：钻铆区域设置FOD围栏，人员进出需通过FOD检查站**
- 碎屑收集：每个钻铆工位配备自动吸尘系统
- **约束：铆接产生的铝屑必须实时收集，禁止散落至复合材料区域**
- 急停按钮：每台钻铆机4个急停按钮（均可达范围≤2m）

## 8. 天车使用要求

- 壁板组件上下型架需使用10t天车
- **约束：天车吊装时，吊装区域下方不可有人员**
- **约束：壁板组件中心需在天车行程范围内，偏移量≤0.5m**
