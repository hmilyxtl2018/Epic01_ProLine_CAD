# 机翼前缘蒙皮铣削工艺规程 SOP-WLE-001-A

**文件编号**: SOP-WLE-001-A  
**版次**: Rev.3  
**生效日期**: 2025年8月15日  
**适航批准**: CAAC-AP-2025-WLE-001  
**NADCAP认证**: 铣削工艺 AC7126  

---

## 1. 适用范围

本规程适用于C919机翼前缘蒙皮零件（件号：WLE-SK-001~012）的数控铣削加工。适用于铝合金2024-T3材料，厚度范围2.0mm~4.5mm。本规程覆盖从毛坯装夹到终检出库全流程。

## 2. 引用标准

- HB 5469-1991 铝合金化学铣切质量检验
- GJB 2639A 航空结构件数控加工通用技术要求
- AS9100D 航空航天质量管理体系

## 3. 设备要求

### 3.1 主设备
- 五轴数控铣床（DMG MORI DMU 125P 或同等级）
- 行程要求：X≥2000mm, Y≥1500mm, Z≥500mm
- 主轴转速范围：500~24000 RPM
- **定位精度：≤0.01mm**
- **重复定位精度：≤0.005mm**

### 3.2 辅助设备
- 真空吸盘工装（工装号：JIG-WLE-V01），吸附力≥500N/m²
- 在线测量测头（Renishaw OMP60）
- 刀具预调仪

## 4. 环境要求

- **环境温度**：20°C ± 3°C
- **环境湿度**：≤70% RH
- 车间洁净度：ISO 8级（Class 100,000）
- **与复合材料加工区的最小隔离距离：15m**（防止金属切屑污染复合材料）
- 光照度：≥500 lux
- 振动等级：VC-C（振动速度≤12.5 μm/s）

## 5. 材料要求

- 铝合金 2024-T3，符合AMS-QQ-A-250/4
- 入库检验：材质证明书+硬度抽检（HB 120~150）
- **来料状态：裸铝，未做表面处理**（表面处理在铣削后工序进行）

## 6. 工艺流程

### 6.1 装夹
1. 将蒙皮毛坯放置于真空工装（JIG-WLE-V01）上
2. 开启真空泵，确认吸附压力：-0.06MPa ~ -0.08MPa
3. 用百分表检查零件平面度
4. **约束：零件装夹变形量≤0.5mm，否则需调整吸盘布局**

### 6.2 粗铣
1. 安装Φ16硬质合金立铣刀（涂层：AlTiN）
2. 参数设置：
   - 切削速度 Vc = 300 m/min
   - 每齿进给 fz = 0.08 mm/tooth
   - 轴向切深 ap ≤ 2.0mm
   - 径向切深 ae ≤ 8mm（50% D）
3. 全程水溶性切削液冷却（浓度5%~8%）
4. **约束：粗铣后需保留精加工余量0.3mm（单边）**

### 6.3 精铣
1. 换装Φ10球头铣刀（微晶硬质合金）
2. 参数设置：
   - 切削速度 Vc = 400 m/min
   - 每齿进给 fz = 0.04 mm/tooth
   - 行距 ae ≤ 0.3mm
   - 残留高度 ≤ 0.005mm
3. **约束：最终表面粗糙度 Ra ≤ 1.6μm**
4. **约束：壁厚公差 ±0.1mm**

### 6.4 检测
1. **在线测量**（机床测头）：关键尺寸公差 ±0.05mm，每件检测
2. **离线CMM测量**：全尺寸检测
   - 首件：100%全检
   - 批次件：30%抽检
   - **约束：CMM检测必须在恒温室（20°C ± 1°C）中进行**
   - **约束：零件需在恒温室内静置不少于4小时后方可测量**

### 6.5 表面质量检查
- 用10倍放大镜目视检查，不允许有裂纹、刀痕、烧伤
- **约束：若发现烧伤，该零件需重新评审（MRB处理）**

## 7. 后续工序衔接

1. 铣削完成 → 去毛刺：**零件铣削结束后2小时内必须完成去毛刺**（防止毛刺氧化嵌入）
2. 去毛刺 → 清洗 → 转化铣车间
3. **约束：零件铣削后裸露铝面在72小时内必须完成阳极氧化处理**（防止晶间腐蚀）
4. **约束：零件在工序间转运必须使用防护托盘，不可直接接触钢制工具或台面**

## 8. 刀具管理

- 每把刀具加工前必须在预调仪上校准
- **约束：刀具磨损量达到VB=0.2mm时必须更换**
- 刀具寿命记录在MES系统中，可追溯至每架次零件

## 9. 质量记录

- 加工过程参数实时记录至MES
- 每件零件附带质量跟踪卡，包含：
  - 材料批号、机床编号、操作员工号、刀具编号
  - 关键尺寸实测值
  - 检测日期和检测员签章
