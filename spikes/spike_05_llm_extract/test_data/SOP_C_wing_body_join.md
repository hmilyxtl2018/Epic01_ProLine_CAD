# 总装翼身对接工艺规程 SOP-WBJ-003-C

**文件编号**: SOP-WBJ-003-C  
**版次**: Rev.4  
**生效日期**: 2025年12月1日  
**适航批准**: CAAC-AP-2025-WBJ-003  
**工艺类别**: 适航冻结工艺（Frozen Process）  

---

## 1. 适用范围

本规程适用于C919飞机最终总装线（FAL）中机翼与机身的对接装配。覆盖左右机翼对接、中央翼盒连接、二次结构安装和测量验收全过程。

**注意：本工艺已通过CAAC适航审查，任何修改需经适航当局重新批准。**

## 2. 引用标准

- HB 8583 飞机大部件对接装配要求
- COMAC ABP-001 C919总装工艺标准
- ISO 10360 坐标测量机验收与复检

## 3. 设备要求

### 3.1 翼身对接工装系统
- 翼身对接型架（工装号：JIG-WBJ-FAL-01）
- **尺寸：40m × 8m × 8m（长×宽×高）**
- **总重：45吨（需30t天车吊装就位）**
- 定位销精度：±0.02mm
- 具备6自由度调姿能力

### 3.2 测量系统
- 激光跟踪仪 × 4台（Leica AT960, 覆盖60m范围）
- 室内GPS（iGPS）× 12个发射器
- **约束：测量系统需在使用前24小时预热和标定**
- **约束：激光跟踪仪工作环境振动 ≤ 0.02mm（振动速度峰值）**

### 3.3 天车系统
- 需占用 30t 桥式天车 × 2台（同步吊装）
- **约束：翼身对接期间，天车不可同时服务其他站位**
- **约束：天车轨道范围必须完全覆盖对接工位（40m×10m区域）**

## 4. 环境要求

- **环境温度**：20°C ± 2°C（高于通常要求，确保测量精度）
- **环境湿度**：40% ~ 60% RH
- **约束：对接全程禁止厂房内有大型物流作业（AGV大件转运），避免地面振动**
- **约束：对接站位（Station-2）上方和两侧15m范围内禁止有其他在建飞机同时进行钻铆作业**（振动隔离）
- 照明：≥1000 lux（对接区域增强照明）
- **约束：对接期间厂房大门（铰链门）必须关闭**（温度稳定性和FOD控制）

## 5. 工艺流程

### 5.1 准备阶段（T-24h）
1. 检查机翼交付状态：完整性检查清单（CL-WBJ-PREP-01）
2. 检查机身准备状态：中央翼盒安装到位
3. 设置测量基准：
   - 在机身上建立4个基准点（激光反射靶标）
   - 在机翼根部建立3个基准点
4. **约束：翼身对接前，机翼和机身的制造偏差报告必须完成且签字确认**

### 5.2 机翼吊装（Day 1, 约4h/侧）
1. 使用30t天车（双钩同步吊装）将机翼从存储架取出
2. 低速水平移动至机身对接位置上方
3. 缓慢下降，导入定位导轨
4. **约束：吊装速度 ≤ 0.5m/min（垂直） / ≤ 1.0m/min（水平）**
5. **约束：吊装过程中机翼最低点距任何障碍物 ≥ 300mm**
6. **约束：先对接左翼，再对接右翼**（结构应力释放顺序）

### 5.3 对接调姿（Day 1~2, 约12h/侧）
1. 使用液压千斤顶进行6自由度调姿
2. 实时激光跟踪仪测量关键配合面间隙：
   - 前梁对接面间隙
   - 后梁对接面间隙
   - 上下壁板对接面间隙
3. **约束：所有对接面间隙 ≤ 0.5mm**
4. **约束：翼尖高度左右差 ≤ 3mm（翼展35m条件下）**
5. 达到公差后插入定位销临时锁定

### 5.4 永久连接（Day 2~3, 约16h/侧）
1. 按图纸钻孔（从型架定位孔引导）
2. 安装高锁螺栓（Hi-Lok）+ 干涉配合铆钉
3. 连接规格：
   - Hi-Lok螺栓 Φ6.35（1/4") × ~200个/侧
   - 干涉铆钉 Φ5.0 × ~800个/侧
4. **约束：高锁螺栓扭矩 = 标准扭矩 ± 10%，每个螺栓扭矩值需记录**
5. **约束：干涉量 = 0.025mm ~ 0.076mm（铆钉与孔的直径差）**

### 5.5 密封与保护
1. 对接缝处涂敷PR-1750 A-2航空密封胶
2. **约束：密封胶涂敷后48小时内不可对接缝施加结构载荷**
3. 涂敷低VOC底漆保护暴露金属面

### 5.6 质量检测
1. 蓝光三维扫描：对接区域外形轮廓度检测
2. **约束：对接区域外形偏差 ≤ 1.5mm（相对理论外形）**
3. X射线检测：关键螺栓连接区域（30%抽检）
4. **约束：X射线检测期间，对接站位15m半径内为辐射禁区**
5. 超声检测：100%覆盖干涉配合铆钉区域
6. 所有NDT结果纳入适航文件包

### 5.7 脉动转运
1. 翼身对接完成后，飞机由AGV转运至Station-3
2. **约束：转运前必须拆除所有临时工装和支撑**
3. **约束：AGV转运速度 ≤ 0.05m/s（3m/min），转运路径净宽 ≥ 10m**
4. **约束：转运过程中飞机不得承受超过0.5g的加速度**

## 6. 特殊情况处理

### 6.1 间隙超差
- 若对接面间隙 > 0.5mm 但 ≤ 1.0mm：使用垫片补偿（需工程批准）
- 若对接面间隙 > 1.0mm：停工，返回上游工序返修
- **约束：垫片使用需获得结构应力工程师和适航代表双签批准**

### 6.2 钻孔偏差
- 孔位偏差 ≤ 0.3mm：接受
- 孔位偏差 > 0.3mm：扩孔至下一规格（需工程审批）
- **约束：同一连接区域扩孔比例不超过10%**

## 7. 工时与产线节拍

- 标准工时：左翼对接 36h + 右翼对接 36h = **72h（3天/架）**
- **约束：翼身对接是脉动线节拍瓶颈，任何延误直接影响年产量**
- 人员配置：每班12人（装配工8人+质检2人+测量2人），三班两运转

## 8. 质量记录与适航

- **约束：所有扭矩值、间隙测量值、NDT报告必须在飞机交付前归档至电子适航文件系统**
- 记录保存期限：飞机全寿命期+退役后2年
- 任何偏差需开具NCR（不符合项报告），经MRB评审

**刻意埋入的矛盾对**（供矛盾检测测试）:
1. **约束WBJ-C08 vs WBJ-C22**: §4 说"对接期间禁止AGV大件转运"，但 §5.7 说"对接完成后AGV转运至下一站"——这不是真正矛盾（时序不同：对接中 vs 对接完成后），但LLM可能误判
2. **约束WBJ-C05 vs 隐含**: §4 要求 20±2°C，但 §4 也说"对接期间大门必须关闭"——如果夏季室外温度高，关门可能导致室内温升超标——这是一个**隐含矛盾**
3. **约束WBJ-C03 vs WBJ-C13**: 天车独占 + 先左翼后右翼——如果只有一台30t天车，则两侧对接必须串行，直接影响72h节拍——这是一个**资源约束冲突**
