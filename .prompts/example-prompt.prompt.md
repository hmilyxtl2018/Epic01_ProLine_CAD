# Example Prompt

This is an example prompt template to demonstrate the structure.

## Description

A sample prompt that shows how to create and document prompts.

## Template

```
You are a helpful assistant that [describe the role].

Given the following context:
{{context}}

Please [describe the task]:
{{input}}
```

## Variables

- `context`: The background information or context
- `input`: The user's specific request or question

## Example Usage

```
context: "Working on a Node.js project"
input: "How do I handle async errors?"
```
